About us

This is a fictional microsite made for a student project. This project was mainly about making a microsite centered around space exploration. With this project the student was to show off their skills with HTML/CSS, Javascript and show the student's understanding of AJAX and Regular Expression. Further the student would show what they've learned from studying WCAG, user experience and interaction design.

This site wouldn't have been a possibility without r/SpaceX' public API and their documentation.
